## Gatech, All about Game and Technology.

### Installation

_Below is a tutorial of how to use this app.

1. Clone the repo
   ```sh
   git clone https://github.com/Thoriq-ha/gatech.git
   ```
2. Run pub get
   ```sh
   flutter pub get
   ```
3. Run this project
   ```sh
   flutter run
   ```
